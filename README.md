## ðŸ“Œ Featured Concept: Inheritance
In the [Java Inheritance I](https://www.hackerrank.com) challenge, I explored how a **subclass** derives attributes and behaviors from a **superclass**.

### Key Learnings:
* **The `extends` Keyword:** Used to create an "IS-A" relationship (e.g., a `Bird` is an `Animal`).
* **Method Reuse:** The `Bird` class automatically has access to the `walk()` method from `Animal` without rewriting it.
* **Specialization:** While inheriting general traits, the subclass can define its own unique behaviors like `fly()` and `sing()`.
